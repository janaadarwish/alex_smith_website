# 👨‍💻 Alex Smith — Web Developer Portfolio

A clean, responsive personal portfolio website built for a fictional web developer, **Alex Smith**. This project was created as a front-end practice exercise using pure HTML and CSS.

---

## 🛠️ Built With

- **HTML5** — Semantic structure and layout
- **CSS3** — Styling, typography, and visual design

---

## 📁 Project Structure

```
alex_smith_website/
├── index.html       # Main HTML page
├── style.css        # Stylesheet
└── assets/          # Images and other static assets
```

---

## ✨ Features

- Personal portfolio layout with sections for bio, skills, and contact
- Clean and minimal design
- Fully built with vanilla HTML & CSS — no frameworks or libraries

---

## 🚀 Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/janaadarwish/alex_smith_website.git
   ```
2. Open `index.html` in your browser — that's it!
